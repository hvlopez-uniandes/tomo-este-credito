# ¿Tomo este crédito? — Prototipo de producto (S7)

Vite + React. Misma lógica que `../prototipo-producto-s7.html`, portada a componente.

## Local

```
npm install
npm run dev
```

## Deploy a Vercel

```
npx vercel        # primera vez: pide login e inicializa el proyecto
npx vercel --prod # despliegue a producción
```

O conecta el repo a Vercel desde vercel.com/new (requiere subir esta carpeta a GitHub primero).
